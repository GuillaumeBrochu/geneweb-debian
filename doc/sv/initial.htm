GeneWeb - Komma igång


Du kan starta GeneWeb på två sätt:
  • Du kan använda programmet gwsetup. Då behöver du bara starta detta program (om inte din installation redan har gjort det automatiskt), öppna sedan den (lokala) webbadress som gwsetup ger och följ instruktionerna. Det är väldigt enkelt att använda och rekommenderas därför för nybörjare.

  • Annars, om du har en del datorkunskap, kan du köra kommandon i ett DOS-fönster. Detta tillåter dig att kontrollera alla kommandona i detalj och förstå vad som händer, men det är inte rekommenderat om du inte är van vid datorer. Klicka här för att starta på detta sätt.
